# 数据查询工具 (Egg Search)

这是一个基于 Python 和 PySide6 开发的本地桌面应用程序，主要用于根据指定的“尺寸”和“重量”条件，快速筛选和查询对应的数据。

## ✨ 功能特点

- **可视化界面**：使用 PySide6 构建了直观的 GUI 界面，包含输入框、查询按钮、数据表格以及底部的状态统计栏。
- **智能范围匹配**：支持处理复杂的数据范围。当源数据为区间（如 `0.29-0.33`）时，只需在工具中输入单点数值（如 `0.30`），程序即可自动解析并判断是否匹配。
- **自动化数据构建**：内置 `create_excel.py` 脚本，可一键将自带的数据集生成带有居中对齐、表头着色等样式的 Excel (`data.xlsx`) 数据源文件。
- **双条件检索**：支持单独根据尺寸、单独根据重量，或结合尺寸与重量双重条件进行过滤检索。

## 🛠️ 环境依赖

请确保您的系统已安装 Python（项目基于 Python 3.13 开发），并安装了以下依赖库：

```bash
pip install PySide6 openpyxl
```
*(注：`PySide6` 用于构建图形交互界面；`openpyxl` 用于读取和创建 Excel 数据表格。)*

## 🚀 快速开始

### 1. 生成数据源
首次运行程序前，或者需要恢复内置测试数据时，请先运行数据生成脚本。该脚本会在项目目录下生成程序依赖的 `data.xlsx` 文件。
```bash
python create_excel.py
```

### 2. 启动应用程序
运行主程序脚本，即可唤出图形界面：
```bash
python app.py
```

### 3. 使用说明
1. 在界面的**“尺寸”**或**“重量”**输入框中填入你想查询的具体数值（如：尺寸输入 `0.30`，重量输入 `6.0`）。
2. 点击**“查询”**按钮，或者在输入框内直接按下回车键即可执行搜索。
3. 匹配结果会即时呈现在下方的表格中，底部状态栏会显示当前匹配到的数据条数。
   - *提示：如果两个输入框都为空，程序会弹出警告要求至少输入一个条件。*

## 📂 核心文件结构

- `app.py`: 应用程序入口与核心逻辑，包含了范围解析函数 `parse_range()`、数据加载函数 `load_data()` 以及 `MainWindow` 界面类的实现。
- `create_excel.py`: Excel 生成脚本，内置了包含“鸭吉吉”、“板板壳”等大量测试数据。
- `.idea/`: PyCharm IDE 的项目工程配置文件夹。

## 📝 数据格式要求
如果您想手动修改或扩充 `data.xlsx` 文件，请确保工作表的第一行（表头）以及数据列遵循以下顺序：
- **第一列**：序号 (如 `011`)
- **第二列**：名称 (如 `鸭吉吉`)
- **第三列**：尺寸 (支持单点如 `0.38` 或区间如 `0.29-0.33`)
- **第四列**：重量 (支持单点如 `6.727` 或区间如 `5.606-6.345`)
数据加载模块会自动从第二行开始读取这些文本并解析为查询条件。
